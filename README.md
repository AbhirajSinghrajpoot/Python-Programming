# 🐍 Python 100 Days of Code Challenge

Welcome to my Python Programming repository! This space documents my journey of mastering Python by committing to the **#100DaysOfCode** challenge. 

I am following the highly acclaimed **"100 Days of Code: The Complete Python Pro Bootcamp" by Angela Yu**. My goal is to code daily, build 100 real-world projects, and level up my programming logic from beginner to advanced.

## 🎯 Course & Challenge Goals
Through this 100-day journey, I will be building a diverse portfolio of projects covering:
- **Core Python:** Scripting, OOPs, File Handling, APIs
- **Web Development:** Backend development using Flask
- **Automation:** Selenium, Web Scraping, Auto-bots
- **Data Science & GUI:** Pandas, Matplotlib, Tkinter

## 📂 Repository Structure
