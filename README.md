Welcome to my Python Programming repository! This space documents my day-by-day journey of mastering Python by committing to the **#100DaysOfCode** challenge. 

I am strictly following the highly acclaimed *"100 Days of Code: The Complete Python Pro Bootcamp"* by Dr. Angela Yu. My goal is to code daily, build 100 real-world projects, and level up my programming logic from beginner scripting to advanced full-stack web development.

---

## 🎯 Course & Challenge Goals

Through this 100-day journey, I am building a diverse portfolio of projects covering:
* **Core Python:** Scripting, Data Types, OOPs, File Handling, Error Handling.
* **Graphical User Interfaces (GUI):** Desktop apps using Tkinter and Turtle graphics.
* **APIs & Automation:** Working with REST APIs, Web Scraping (BeautifulSoup), and browser automation (Selenium).
* **Web Development:** Building robust backends using **Flask**, Jinja templates, and database integration (SQLite/SQLAlchemy).
* **Data Science:** Data exploration and visualization using Pandas and Matplotlib.

The bootcamp is structured into distinct phases of mastery. Here is how my repository is organized as I progress:

Building full-stack web applications from scratch.

Tackling advanced, capstone-level projects with minimal guidance to prove my mastery.

## 🚀 How to Run the Projects
To run any of the daily projects on your local machine:
 1. **Clone this repository:**
   ```bash
   git clone [https://github.com/AbhirajSinghrajpoot/Python-Programming.git](https://github.com/AbhirajSinghrajpoot/Python-Programming.git)
   
   ```
 2. **Navigate to the specific day's folder:**
   ```bash
   cd "Day 1" 
   
   ```
 3. **Run the script:**
   Ensure you have Python 3.x installed.
   ```bash
   python main.py
   
   ```
*(Note: Advanced projects in the later days may require installing dependencies via pip install -r requirements.txt. These will be noted in the specific project folders.)*
